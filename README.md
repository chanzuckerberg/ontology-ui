### Usage

The following commands will run a python/flask server on 5000, and a vite client dev server on 3000.

##### CLIENT

```
$ npm ci
$ npm run start
```

##### SERVER

Setup:

```
$ cd server
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
```

Run:

```
$ flask run
```

And navigate to `http://localhost:3000/`
